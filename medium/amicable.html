Problem 1: Amicable Numbers

Background

The proper divisors of a number all of the the divisors of that number except the number itself. For example, the proper divisors of 6 are (1, 2, 3) and the proper divisors of 12 are (1, 2, 3, 4, 6).

Two numbers are said to be amicable if the sum of each number's proper divisors is the other number. For example, (220, 284) are amicable numbers: 

        220: (1, 2, 4, 5, 10, 11, 20, 22, 44, 55, 110): sum = 284
        284: (1, 2, 4, 71, 142): sum = 220

Input Expectations

Each line of text is a pair of integers. The number of inputs may vary. Please name your input file "1in.txt"

Sample Input

220 284
1184 1210
3234 3425

Output Expectations

There will be one line of output for each line of input. The output for each line must be either "amicable pair" or "NOT amicable pair"

Sample Output

amicable pair
amicable pair
NOT amicable pair
