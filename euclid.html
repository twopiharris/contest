Problem 5: Euclid

Background

Euclid discovered an ingenious mechanism for solving the greatest common divisor between two numbers thousands of years ago. Your mission is to write a program that uses his algorithm to efficiently find the greatest common divisor (GCD).

Here is the basic version: 


Given two divisors (x and y)
If x is less than y, swap x and y
as long as y is not zero:
    find the remainder of x / y
    copy y to x
    copy the remainder to y
    
The last non-zero remainder will be the GCD

Input Expectations

Each line of input will consist of two integer divisors. They might or might not be largest integer first. The number of inputs is not known.

Please use "5in.txt" as the input file name.

Sample Input

54 24
1785 546

Output Expectations

Every line of input should result in one line of output. The output will be the greatest common divisor of the inputs.

Sample Output

6
21
